# Font Pairings — Premium Web 2025-2026

A reference of typeface combinations observed across recent premium sites. The goal: when picking type for a project, have a tested pairing as starting point, then deviate with intent.

## The recurring faces

1. **Editorial New** (Pangram Pangram) — serif, modern, editorial. Free for commercial.
2. **GT Sectra** (Grilli Type) — serif, sharp, contrasted. Premium license required.
3. **Canela** (Commercial Type) — serif, soft, distinctive. Premium.
4. **GT Super** (Grilli Type) — serif, sharp. Premium.

7. **Aeonik** (CoType) — sans, neo-grotesque. Premium.

9. **Druk** (Commercial Type) — heavy sans, condensed. Premium.

## Free vs paid recommendation

For 80% of projects, the free stack works:
- **Geist Sans + Geist Mono + Instrument Serif** (italic accent) covers editorial-to-tech projects.
- **Inter + JetBrains Mono + Newsreader** (body serif) covers SaaS and content-led projects.

When to license premium type:
- Brand-critical projects where typography is the differentiation.
- Agency projects where the type is the visual signature.
- Heritage / luxury projects where free type undermines positioning.
- When the brand identity uses a specific licensed face.

## Typeface combinations to avoid

These pairings are AI-default or visually weak:

- **Inter + Inter** (no variation) — flat, the default of defaults.
- **Inter + Roboto** — too similar, no contrast.
- **Playfair Display + Roboto** — the wedding website default.
- **Montserrat + anything** — overused for "modern" branding.
- **Open Sans + Lato** — 2014 corporate default.
- **Poppins + anything** — Google Fonts overuse.
- **Bebas Neue + anything** — the YouTube thumbnail default.
- **Two display faces together** — pick one display and stick with it.
- **Three different sans serifs** — pick one, vary weight/size for hierarchy.

## Variable fonts

Variable fonts are an underused superpower in 2025-2026. They allow:
- Smooth weight transitions on hover.
- Optical sizing (different weights at different sizes).
- Custom axes for character variations.
- Animation between weights.

**Recommended free variable fonts:**
- **Inter Variable** — covers weight, slant axes.
- **Geist Sans Variable** — weight axis.
- **Recursive** — multiple axes including casual-formal.
- **Fraunces** — sophisticated serif with SOFT and WONK axes.

**Use cases:**
- Hover state: animate weight from 400 to 600 on a button (smoother than swapping faces).
- Headlines: extreme weights for editorial impact.
- Responsive: lighter weight at larger sizes, heavier at small sizes.

## How to set type on the web

A separate concern from picking faces. Key rules:

- **Font display: swap** — never let the page load without text.
- **Preload critical fonts** — `<link rel="preload" as="font">`.
- **Limit faces** — 2-3 maximum per project. Variable fonts count as one.
- **Self-host where possible** — Google Fonts CDN has privacy and performance trade-offs.
- **Subset** — only load characters you need (LATIN-EXT for French, basic LATIN for English-only).
- **Fallback stack** — define system fallbacks that match metrics (`font-size-adjust`, `size-adjust`).

```css
@font-face {
  font-family: 'Geist Sans';
  src: url('/fonts/geist-sans-variable.woff2') format('woff2-variations');
  font-weight: 100 900;
  font-display: swap;
}

:root {
  --font-sans: 'Geist Sans', -apple-system, BlinkMacSystemFont, 'Segoe UI',
               sans-serif;
}
```

## Final note

Typeface choice is the single most legible signal of design quality. Most "premium" projects achieve premium status primarily through type. Spend disproportionate effort here. A budget that excludes type licensing is a budget that doesn't take the project seriously.
